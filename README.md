# media-transfer
Transfer Spotify playlists to YouTube Music


## Create a virtual environment
python3 -m venv media-transfer

### Activate the virtual environment
source media-transfer/bin/activate  # On Linux/macOS
media-transfer\Scripts\activate     # On Windows

### Install a package within the virtual environment
pip install

### Deactivate the virtual environment
deactivate

## .env variables

```
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_USER_ID=
```
